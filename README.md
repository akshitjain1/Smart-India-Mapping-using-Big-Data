# Smart India Mapping using Big Data

## Overview

This project focuses on analyzing and visualizing air pollution data of major North Indian cities using big data techniques. The goal is to understand pollution trends and represent them visually on maps.

## Features

* Data cleaning and preprocessing using Pandas
* Analysis of PM10 pollution levels
* Visualization using Matplotlib
* Interactive map using Folium
* Basic Computer Vision using OpenCV

## Dataset

Dataset is taken from data.gov.in:
https://www.data.gov.in/resource/status-big-cities-north-india-2013-2015fromministry-environment-forest-climate-change

## How to Run

1. Create virtual environment
2. Install dependencies:
   pip install -r requirements.txt
3. Run the project:
   python app.py

## Outputs

* Graph of pollution levels
* Interactive India map
* Edge-detected image (Computer Vision)

## Limitations

* Dataset is limited to selected cities
* Coordinates are manually added
* Basic CV implementation (can be improved)

## Conclusion

This project demonstrates how big data and visualization techniques can be used to analyze environmental data and support better decision-making.
